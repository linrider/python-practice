# 01 — Syntax & Functions

Навчальні вправи для тренування базового Python.

## Задача: Word Statistics

CLI-програма, яка аналізує текстовий файл.

### Можливості

Програма:

* читає текстовий файл
* рахує кількість рядків
* рахує кількість слів
* показує топ найчастіших слів

### Нормалізація тексту

Перед підрахунком текст:

* переводиться у lowercase
* очищується від пунктуації
* розбивається на слова

### Що тренувалось

* функції
* робота з файлами
* стандартна бібліотека (`collections.Counter`)
* обробка аргументів CLI (`argparse`)
* `pathlib`
* нормалізація тексту
* базова структура Python CLI-програми

### Запуск

```bash
python word_stats.py sample.txt
```

або

```bash
python word_stats.py sample.txt --top 5
```

### Приклад результату

```
Lines: 4
Words: 13

Top words:
python: 3
is: 3
powerful: 2
simple: 2
```
