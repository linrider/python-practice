# 01 — Syntax & Functions

Навчальна задача для тренування базового синтаксису Python.

## Задача: Word Statistics

Програма читає текстовий файл і рахує:

* кількість рядків
* кількість слів
* топ найчастіших слів

## Що тренується

* функції
* робота з файлами
* стандартна бібліотека (`collections.Counter`)
* обробка аргументів CLI (`sys.argv`)
* базова структура Python-програми

## Запуск

```bash
python word_stats.py sample.txt
```

## Приклад результату

```
Top words:
python: 3
is: 3
powerful: 2
simple: 2
great: 1
things: 1
are: 1
```
