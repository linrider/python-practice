# 02 — Log Analyzer

CLI-програма для аналізу лог-файлу.

## Що робить

- читає лог-файл построчно
- рахує кількість записів за рівнями логування
- показує статистику для `INFO`, `WARNING`, `ERROR`

## Що тренується

- читання файлу построчно
- робота з `Counter`
- розбір рядків
- CLI-аргументи

## Запуск

```bash
python core/02_logs/log_analyzer.py core/02_logs/sample.log
